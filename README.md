# 🎵 Spotify Clone

A Spotify-inspired music player built using **HTML, CSS, and JavaScript**. This project recreates the core functionality of a music player, allowing users to browse playlists, select songs, and control audio playback through an interactive interface.

## 🚀 Features

- 🎧 Browse and select playlists
- ▶️ Play and pause songs
- ⏭️ Skip to the next track
- ⏮️ Return to the previous track
- 🔀 Random song selection
- 🎵 Display the currently playing song
- 📱 Responsive and clean user interface inspired by Spotify

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
